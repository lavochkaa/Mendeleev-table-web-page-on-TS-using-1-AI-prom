# Интерактивная таблица Менделеева — Инструкция по запуску

![Предпросмотр](public/preview.png)

---

## Что это такое

3D-визуализация всех 118 химических элементов с анимированными атомными орбиталями.
При нажатии на элемент открывается панель с атомной структурой, электронными оболочками и свойствами.

---

## Что нужно установить (один раз)

### 1. Node.js

Перейдите на сайт **https://nodejs.org** и скачайте версию **LTS** (рекомендуемая).
Установите как обычную программу, нажимая «Далее».

Чтобы проверить, что установилось — откройте **Терминал** и введите:
```
node --version
```
Должно появиться что-то вроде `v20.11.0`.

### 2. Скачать проект

Если у вас установлен Git:
```
git clone https://github.com/lavochkaa/Mendeleev-table-webpage
cd Mendeleev-table-webpage
```

Или просто скачайте архив ZIP с GitHub (зелёная кнопка **Code → Download ZIP**), распакуйте в удобное место.

---

## Запуск

Откройте **Терминал**, перейдите в папку с проектом:

```bash
cd путь/до/папки/с/проектом
```

> Совет: можно перетащить папку прямо в окно Терминала — путь вставится автоматически.

Затем выполните по очереди:

```bash
npm install
```
*(ждёт 1–2 минуты, скачивает библиотеки)*

```bash
npm run dev
```

Откройте браузер и перейдите по адресу:

**http://localhost:3000**

## Управление

| Действие | Как |
|----------|-----|
| Переключить расстановку | Клавиши **1–9** или кнопки внизу экрана |
| Вращать таблицу | **Зажать мышь + тянуть** |
| Приблизить / отдалить | **Колёсико мыши** |
| Открыть карточку элемента | **Кликнуть** на элемент |
| Развернуть атом на весь экран | **Кликнуть** на модель атома в карточке |
| Остановить анимацию орбит | Кнопка **⏸** на атоме |
| Закрыть карточку | Кнопка **✕** или клик вне панели |
| Звук вкл/выкл | Клавиша **M** или кнопка звука |

## Расстановки (клавиши 1–9)

| Клавиша | Название | Описание |
|---------|----------|----------|
| 1 | Таблица | Классическая таблица Менделеева |
| 2 | Сфера | Элементы на поверхности сферы |
| 3 | Спираль | Одиночная спираль |
| 4 | Куб | Трёхмерная сетка 5×5×5 |
| 5 | Галактика | Спираль галактики |
| 6 | ДНК | Двойная спираль ДНК |
| 7 | Атом | По электронным оболочкам |
| 8 | Нейросеть | Облако нейросети |
| 9 | Волна | Волновая поверхность |

## Если что-то не работает

- Убедитесь, что команда `npm install` выполнилась без ошибок
- Попробуйте другой браузер (лучше всего работает **Google Chrome**)
- Перезапустите терминал и повторите `npm run dev`
